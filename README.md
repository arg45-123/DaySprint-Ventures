# Team: DaySprint-Ventures
Problem statement: RozgarSaathi
Domain: App Development 
objectives: 
1) Provide digital visibility to daily wage workers
2) Enable quick job discovery based on location
3) Ensure direct communication between workers and employers
4) Reduce idle time and increase daily earnings
Features & Functionality
1) Skill-based worker registration (plumber, electrician, painter, etc.)
2) Geo-based job feed (nearby jobs and workers)
3) WhatsApp-style communication flow
4) Real-time job matching system
5) No middleman, no commission
Tech Stack Used
Frontend:  Flutter
Backend:  Firebase
Database:  Firestore
Authentication: OTP-based login
APIs: Google Maps API (for location-based services)
Setup/Installation Instruction
Flutter, dart language , firbase for backend
